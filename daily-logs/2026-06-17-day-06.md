## Key takeaway
Knowing how packages get installed on a Linux system is a real incident response skill — `apt list --installed` or `dpkg -l` lets you audit what's running on a host and spot something that shouldn't be there. Attackers sometimes hide malicious tools as legitimate-looking packages, so this knowledge directly supports detecting persistence mechanisms later in my SOC training.
